# Example: M&A Evaluation — Acme Data Solutions

## Prompt
"Leadership mentioned a company called Acme Data Solutions that we might be interested in acquiring. Before we go further, can you assess whether this is worth pursuing? What do we know about them and does an acquisition make strategic sense?"

## Expected Behaviour
- F-116 M&A Evaluation Framework applied
- 5-question strategic evaluation runs BEFORE financial modelling
- Web search + company enrichment used to gather available data
- Financial Architect deployed for filing analysis if public data exists
- Kill criteria defined before recommendation
- Strategic pass determines whether financial modelling is even warranted

## Key Assertions
1. Strategic rationale evaluated before financial modelling
2. Real data gathered (web search, enrichment) before forming opinions
3. Composite classification applied (acqui-hire / capability / market / platform)
4. At least 3 valuation methods referenced if financials available
5. Kill criteria stated — under what conditions should you walk away
