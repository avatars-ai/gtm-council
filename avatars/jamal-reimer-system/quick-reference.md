# Jamal Reimer — Quick Reference Card

## Use This In Live Deal Conversations

### Two Mountain Model — Quick Check
| Mountain | Question | Status |
|---|---|---|
| **M1: Pain** | Is the business problem quantified in $ terms? | Yes / No / Partial |
| **M1: Value** | Is our solution mapped to their specific metrics? | Yes / No / Partial |
| **M1: ROI** | Is the ROI model co-built with the customer? | Yes / No / Partial |
| **M1: Risk** | Have we addressed "what if it doesn't work?" | Yes / No / Partial |
| **M2: Exec Sponsor** | Do we have a C-level champion? | Identified / Engaged / Aligned / None |
| **M2: Champion** | Does our champion have Power + Access + Vested Interest? | All 3 / Missing 1+ |
| **M2: Blockers** | Have we identified who blocks this and why? | Yes / No |
| **M2: Personal Wins** | Do we know what each stakeholder gains personally? | Yes / No / Partial |
| **M2: Consensus** | Can the buying group reach agreement? | Yes / At Risk / No |

### Pipeline Flywheel — Quick Check
| Phase | Question | Status |
|---|---|---|
| **Selection** | Did we choose this account strategically (not randomly)? | Strategic / Opportunistic |
| **Entry** | Are we multi-threaded (3+ relationships)? | Yes / No |
| **Value** | Have we delivered insight that changed their thinking? | Yes / No |
| **Referral** | Will this deal generate 2-3 warm intros for next deals? | Yes / No / Unknown |

### Deal Architecture — Quick Sizing
- **Is the deal sized to the full problem scope?** If not, why are we self-limiting?
- **What does Year 2 look like?** If you can't describe it, you haven't architected the deal.
- **Is the pricing model aligned with how they get value?** (Seats? Usage? Outcomes?)
- **Are contract terms designed for expansion?** (Price locks? Volume incentives? Multi-year?)

### Top 5 Diagnostic Questions (Use in Any Deal Review)
1. "Which mountain is incomplete — business case or political case?"
2. "Who is the executive sponsor and what do they gain personally?"
3. "If you left the room, who sells this deal internally? That's your champion."
4. "What does this deal look like in Year 2? Year 3?"
5. "Who loses if this deal goes through? Have you addressed them?"

### Deal Variable Levers (Quick Reference)
| Variable | Pull This Lever When... |
|---|---|
| **Scope** | Deal is undersized relative to the problem |
| **Timeline** | Customer is stalling without urgency |
| **Contract length** | Customer wants better pricing (trade for commitment) |
| **Exec involvement** | Decision is stuck at middle management |
| **Phasing** | Customer is risk-averse (prove value first, expand later) |
| **Champion investment** | Champion is passive (arm them with ammunition) |
| **Pricing model** | Current model doesn't align with customer's value realisation |

### Red Flags (Immediate Action Required)
- Single-threaded (only one relationship in account)
- No executive sponsor identified
- Champion fails Power/Access/Vested Interest test
- Business case not quantified in customer's own metrics
- Buyer group cannot agree on the problem (consensus failure)
- Deal has been at same stage for 30+ days
- "We need to start smaller" without a clear Phase 2 path
- Competitor has executive-level access and you don't
