# Marketing Review Template

*For CMO quarterly business reviews, marketing budget defence, programme retrospectives,
and marketing-to-board communication. Designed for marketing leaders presenting to
CEO, board, or cross-functional executives.*

*Distinct from sales QBR templates — marketing's value is measured differently and the
audience has different expectations. Sales QBRs report on pipeline and revenue.
Marketing QBRs report on demand generation, brand health, and marketing-sourced ROI.*

---

## When to Use

- CMO quarterly business review
- Marketing budget defence / investment case
- Programme or campaign retrospective
- Annual marketing plan presentation
- Marketing contribution to board memo (Section 3 input for `board-memo.md`)

---

## Structure

### 1. Marketing Contribution Summary (3-5 sentences)

What did marketing contribute to the business this quarter? State it in the language
the CEO and CRO speak — pipeline, revenue, efficiency — not marketing jargon.

**Template:** "Marketing [sourced/influenced] $[X]M pipeline ([Y]% of total) and
contributed $[Z]M closed revenue at $[W] CAC. [One sentence on the defining
programme/channel]. [One sentence on what changes next quarter]."

### 2. Attribution Dashboard

The most contested dashboard in any GTM org. Present BOTH sourced and influenced —
because the CRO will ask about sourced and the CEO will ask about total impact.
Presenting only one invites the objection the other answers.

**2a. Pipeline Attribution**

| Source | Pipeline Generated | % of Total | Closed Revenue | CAC | Trend |
|---|---|---|---|---|---|
| Marketing-Sourced | | | | | ↑ ↔ ↓ |
| Marketing-Influenced | | | | | ↑ ↔ ↓ |
| Sales-Sourced | | | | | ↑ ↔ ↓ |
| Partner-Sourced | | | | | ↑ ↔ ↓ |
| Total | | 100% | | | |

**Rules:**
- Sourced = first touch originated by marketing
- Influenced = marketing touched at any point in the deal cycle
- Both are valid. Neither is complete. Present both and let the audience choose their lens.
- If sourced + influenced + sales-sourced > 100%, explain the overlap methodology.

**2b. Channel Performance**

| Channel | Spend | Pipeline | Revenue | CAC | CPL | ROI | Trend |
|---|---|---|---|---|---|---|---|
| Paid Search | | | | | | | |
| Paid Social | | | | | | | |
| Content / SEO | | | | | | | |
| Events / Field | | | | | | | |
| ABM | | | | | | | |
| Email / Nurture | | | | | | | |
| Partner Marketing | | | | | | | |

**Rules:**
- Every channel has a spend number. Channels without attributable spend still have
  team cost — include it.
- ROI = (Revenue - Spend) / Spend. If ROI is negative, say so. Hiding it erodes trust.
- Trend matters more than absolute number. A channel declining at -15% QoQ needs
  attention regardless of current ROI.

### 3. Brand Health (Leading Indicators)

Brand investment is the hardest marketing spend to defend because it doesn't produce
pipeline in the quarter it's spent. These leading indicators give the board
confidence that brand investment is working BEFORE the lagging outcomes arrive.

**Monthly indicators (reported quarterly):**

| Indicator | Q[X] | Q[X-1] | Trend | Target |
|---|---|---|---|---|
| Branded search volume | | | ↑ ↔ ↓ | |
| Direct traffic (% of total) | | | ↑ ↔ ↓ | |
| Share of voice vs competitors | | | ↑ ↔ ↓ | |
| Inbound lead volume (no paid) | | | ↑ ↔ ↓ | |
| SDR connect rate (cold vs warm) | | | ↑ ↔ ↓ | |

**Quarterly indicators:**

| Indicator | Q[X] | Q[X-1] | Trend | Target |
|---|---|---|---|---|
| Win rate: branded vs unbranded leads | | | ↑ ↔ ↓ | |
| Sales cycle: branded vs unbranded | | | ↑ ↔ ↓ | |
| Inbound pipeline velocity | | | ↑ ↔ ↓ | |
| NPS / brand perception survey | | | ↑ ↔ ↓ | |

**Annual indicators:**
- Unaided brand recall in target segment
- Pricing power (average discount rate trend)
- CAC trend (brand reduces CAC over time if working)

**Key principle:** Brand metrics are LEADING indicators. They predict future pipeline
and revenue. If branded search is growing 20% QoQ, pipeline will follow — the lag
is typically 2-4 quarters. Report the lag explicitly.

### 4. Programme Retrospective (Top 3 Programmes)

For each of the quarter's top 3 programmes (by spend or strategic importance):

| Element | Detail |
|---|---|
| **Programme** | [Name] |
| **Objective** | [What it was designed to do] |
| **Spend** | [$X] |
| **Results** | [Pipeline, leads, meetings, registrations — whichever is the primary KPI] |
| **ROI** | [If measurable. If not, state the leading indicator] |
| **Learning** | [One sentence: what did we learn?] |
| **Decision** | Scale / Maintain / Adjust / Kill |

**Rules:**
- Under-spend is as much a problem as over-spend. Budget returned = missed opportunity.
- Ring-fence brand budget with different success criteria than demand gen budget.
  Brand is measured on leading indicators (Section 3), not pipeline.

### 6. Next Quarter: Priorities & Investment Shifts

3-5 bullet priorities with specific, measurable targets:

- **Priority 1:** [What] → [Target metric] → [By when]
- **Priority 2:** [What] → [Target metric] → [By when]
- **Priority 3:** [What] → [Target metric] → [By when]

Investment shifts: which channels/programmes get MORE budget, which get LESS, and WHY.
The "why" must reference data from Sections 2-4 — not intuition.

### 7. Asks (if any)

Same principle as board-memo.md: specific, resourced, time-bound. If no asks, omit.

## Cross-Functional Defence Patterns

Marketing reviews are frequently challenged by sales leadership. These are the
common objections and the framework for responding:

| Challenge | Response Framework |
|---|---|
| "Only 18% is marketing-sourced" | Present influenced pipeline alongside sourced. "18% sourced, 62% influenced. The question is whether you want the 62% to stop." F-100 Dual-Axis. |
| "Your MQLs are garbage" | MQL acceptance rate. If <60%, the MQL definition needs recalibration — that's a joint problem, not a marketing problem. F-100. |
| "Show me pipeline this quarter" | Brand investment compounds. Present the 2-4 quarter lag with leading indicator evidence. Ring-fenced budget with kill gate. |
| "We should cut marketing and invest in outbound" | Present CAC by source. Marketing-sourced deals typically close at 60-70% of sales-sourced CAC. F-107 Revenue per AE. |
| "What do we actually get for $[X]M?" | Channel performance table (Section 2b). Every dollar has a measurable outcome. |

---

## Quality Gates

1. Does the attribution dashboard show BOTH sourced and influenced?
2. Does every channel have a spend number and an ROI?
3. Are brand metrics framed as leading indicators with explicit lag periods?
4. Are programme retrospectives honest — including kills and learnings?
5. Are priorities specific enough to measure next quarter?
6. Could the CRO read this and say "I understand what marketing did for my pipeline"?
