# Board Memo Template

*For quarterly board updates, investor communications, and company-level business reviews.
Distinct from the Cliff Edge structure — board memos are REPORTING not PERSUASION.
The audience has authority over you. They need confidence in your command of the business,
not a sales pitch for resources.*

---

## When to Use

- Quarterly board meetings
- Investor update letters
- Company-level QBRs (CEO/CRO presenting to board or executive committee)
- Annual plan presentations
- Any communication where the audience evaluates YOUR performance, not a deal's

## Structure

### 1. Executive Summary (3-5 sentences)

State the quarter's outcome vs plan, the single most important thing that happened,
and the one thing that matters most going forward. No preamble. No context-setting
for people who already know the context.

**Template:** "Q[X] [year]: [metric] at [result] vs [target] ([delta]%). [One sentence
on the defining event]. [One sentence on what this means for next quarter]."

### 2. KPI Dashboard (5-7 metrics, no more)

Present the metrics that define business health. Board members read the dashboard
FIRST and the narrative SECOND. If the numbers tell a bad story, don't try to
bury it in narrative — they'll find it.

| Metric | Q[X] Actual | Q[X] Target | Q[X-1] Actual | Trend | Commentary |
|---|---|---|---|---|---|
| ARR / Revenue | | | | ↑ ↔ ↓ | One sentence |
| NRR | | | | ↑ ↔ ↓ | |
| New Business Bookings | | | | ↑ ↔ ↓ | |
| Pipeline Coverage (next Q) | | | | ↑ ↔ ↓ | |
| CAC Payback | | | | ↑ ↔ ↓ | |
| Gross Margin | | | | ↑ ↔ ↓ | |
| Headcount / Efficiency | | | | ↑ ↔ ↓ | |

**Rules:**
- Every metric has a target. No metrics without targets — that's a vanity dashboard.
- Every metric has a prior-quarter comparator. Trend matters more than point-in-time.
- Commentary is ONE sentence. If it needs more, it goes in Section 3.
- Never present a metric that looks good without its counter-metric (Rabois paired indicators).

### 3. Narrative: What Happened, What It Means, What We're Doing

**What Happened (3-5 paragraphs max):**
Facts. Ordered by significance, not chronology. Lead with the most important thing,
not the first thing that happened. Each paragraph covers ONE topic.

**What It Means (2-3 paragraphs):**
Interpretation. This is where the board learns whether you understand the business
or are just reporting numbers. Connect the facts to the strategy. Name the risks.
Name the opportunities. The Stockdale Paradox applies: faith you will prevail +
brutal honesty about current reality.

**What We're Doing (2-3 paragraphs):**
Actions already taken and planned. Specific, time-bound, measurable. The board
doesn't want to hear "we're focused on improving pipeline" — they want to hear
"we've launched a 30-day outbound sprint targeting 200 accounts in segment X,
expected to generate $2M pipeline by [date]."

### 4. Risks & Mitigations (table format)

| Risk | Probability | Impact | Mitigation | Status |
|---|---|---|---|---|
| [Specific risk] | High/Med/Low | [Quantified] | [Specific action] | [In progress / Planned / Complete] |

**Rules:**
- Maximum 5 risks. If you have more, the board loses confidence — not because of
  the risks, but because you can't prioritise them.
- Every risk has a mitigation. Unmitigated risks are asks (Section 5), not risks.
- Quantify impact in revenue/ARR/cash terms, not percentages alone.

### 5. Asks (if any)

What do you need from the board? Specific, resourced, time-bound.
If you have no asks, don't include this section — forced asks undermine real ones.

### 6. Appendix (optional)

Detailed data that supports the narrative but would slow the main document.
Referenced from Sections 2-4 but not required to understand the memo.

## Register & Tone

- **Confidence without arrogance.** You command the business. You know the numbers.
  You know what you don't know. You have a plan.
- **Candour without defensiveness.** Bad quarters happen. The board respects leaders
  who name problems early, not leaders who hide them.
- **Precision without verbosity.** Every word earns its place. The board's time is
  the most expensive in the room.
- **No promotional language.** You are not selling the board on the business —
  they already own it. Report, interpret, recommend.

---

## Quality Gates

1. Could a board member scan this in 90 seconds and know whether the quarter was good or bad?
2. Does every metric have a target and a comparator?
3. Are risks quantified in revenue terms?
4. Are asks specific enough to approve in the meeting?
5. Would you be comfortable if this leaked to the press?
